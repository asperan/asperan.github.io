    • register

      Declaration

      pure nothrow @safe ActorSystem register(T : Message)(T delegate(JSONValue) unmarshaller);

      Registers an unmarshalling function for the given Message type.

      Discussion

      The unmarshalling function should be the inverse operation of the implemented Message.toJson method.

      Calling this method again on the same Message type will override the used marshaller, so only the last specified will be used.

      Parameters

      T

      the specific Message type.

      T delegate(JSONValue) unmarshaller

      the unmarshalling function for the message type T.

      Return Value

      the ActorSystem instance.

      const @safe ActorReference getRemoteReference(string host, ushort port);

      Provide a reference to a remote actor.

      Discussion

      Parameters

      the host address.

      ushort port

      the port.

      Return Value

      a reference to a remote actor.

      Throws

      an AddressException if the given host cannot be resolved.
